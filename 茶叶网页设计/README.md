# 茶叶网页设计项目

这是一个精美的茶叶主题响应式网页设计项目，展示了各种茶叶产品及其详细信息。该项目采用现代前端技术开发，具有优雅的界面设计和流畅的用户体验。

## 项目特点

- 响应式设计，适配各种设备屏幕
- 精美的茶叶图片展示
- 详细的产品信息页面
- 简洁明了的导航系统
- 优雅的视觉效果和过渡动画
- 表单验证功能

## 项目结构

```
├── css/             # 样式文件目录
│   └── common.css   # 通用样式
├── images/          # 图片资源目录
│   ├── focus1.jpg   # 焦点图片
│   ├── index1-6.jpg # 首页图片
│   └── view1-8.jpg  # 产品详情页图片
├── js/              # JavaScript文件目录
│   └── validate.js  # 表单验证脚本
├── index.html       # 首页
├── view1-9.html     # 产品详情页
└── README.md        # 项目说明文档
```

## 技术栈

- HTML5
- CSS3
- JavaScript
- 响应式设计

## 使用方法

1. 克隆或下载本项目到本地
2. 使用浏览器打开 `index.html` 文件即可查看网站
3. 点击导航菜单或产品图片进入产品详情页面

## 开发说明

### 样式定制

所有样式文件都位于 `css/` 目录下，您可以通过修改 `common.css` 文件来自定义网站的外观。

### 图片资源

所有图片资源都位于 `images/` 目录下，您可以替换为自己的图片资源，但请注意保持相同的文件名或更新相应的HTML文件中的引用。

### 交互功能

表单验证功能通过 `js/validate.js` 文件实现，您可以根据需要修改验证规则或添加新的交互功能。

## 浏览器兼容性

该项目兼容所有现代浏览器，包括但不限于：

- Chrome 60+
- Firefox 55+
- Safari 11+
- Edge 16+
